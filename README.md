# dice-roller app

A simple test app for dice rolling. This application was used as a playground to test Github Copilot.

Most of the code is automatically generated by the Github Copilot extension.

## ⚙️ Install

```shell
cd dice-roller\

npm install
```

## 🛠️ Dev

```shell
npm run dev
```

## 🚀 Prod

```shell
npm run build
npm start
```
